### About Photoshop


### Photoshop Command

- `Command` + `Alt` + `Shift` + `k` : 단축이 설정 관리
- `Command` + `N` : 새 프로젝트 만들기
- `Command` + `R` : 눈금자 on off
- `Command` + `H` : 가이드라인 on off
- `Command` + `J` : 현재 레이어 복제
- `Command` + `G` : 선택 레이어 그룹화
- `Command` + `Alt` + `I` : 이미지 크기 변경
- `Command` + `Alt` + `C` : 캔버스 크기 변경
- `Command` + `Alt` + `Z` : 실행 취소
- `Command` + `Shift` + `Z` : 다음 실행
