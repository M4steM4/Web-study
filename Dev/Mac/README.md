### Mac Command

- `command` + `option` + `i` : 개발자도구 띄우기
- `command` + `option` + `v` : 잘라내기
- `command` + `1/2/3/4` : `탭이동`
- `command` + `tap` + `w` : 탭에서 간편삭제
- `command` + `option` + `d` : 독 내리기
- `command` + `option` + `h` : 현재 창 이외의 다른창 내리기
- `command` + `comma` : setting
