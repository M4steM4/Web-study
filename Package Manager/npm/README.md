### package.json

- `name` : 대문자x
- `version` : 패키지의 고유성 판별
- `description` : 패키지의 설명
- `keywords`: 키워드를 문자열 배열로 설명
- `homepage` : url과 다른 항목
- `bugs` : 버그트레킹, issue email url
- `license` : 패키지 권한 및 사항 설정
- `repository` : type과 url설명
